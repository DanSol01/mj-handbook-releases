## 📌 О приложении

**MJ Handbook** — настольный справочник для сотрудников службы скорой помощи (EMS) roleplay-сервера **Majestic RP**.

Приложение работает **поверх игры** и позволяет мгновенно находить протоколы, коды болезней и служебные инструкции прямо во время игровой сессии — не выходя из игры.

---

## ✨ Возможности

| Функция | Описание |
|---|---|
| 📂 **Документы** | Создавай разделы со статьями — протоколы, инструкции, регламенты |
| 📝 **Заметки** | Свободный текст — тен-коды, позывные, прайс-лист |
| 🔍 **Поиск** | Мгновенный поиск по всему справочнику |
| 📋 **Копирование** | Одна кнопка — текст в буфер, вставляй в чат через Ctrl+V |
| ⌨️ **Горячая клавиша** | Ctrl+Shift+E скрывает и показывает окно поверх игры |
| 🔄 **Автообновление** | Приложение само уведомит о новой версии и обновится |
| 💾 **Экспорт / Импорт** | Резервная копия всех данных в JSON |
| 🔐 **Защита** | Один лицензионный ключ = одно устройство (HWID) |

## ⬇️ Установка

### 1. Скачай установщик

Перейди в раздел [**Releases →**](../../releases/latest) и скачай `MJ Handbook Setup x.x.x.exe`

### 2. Запусти установщик

> ⚠️ **Windows может показать предупреждение SmartScreen** — это нормально для приложений без платного сертификата.
> Нажми **«Подробнее»** → **«Выполнить в любом случае»**

### 3. Введи лицензионный ключ

После установки приложение попросит лицензионный ключ.
За ключом обращайся к руководству EMS в Discord.

## 🔄 Обновления

Приложение автоматически проверяет обновления при каждом запуске.
При наличии новой версии появится баннер с кнопкой **«Обновить»** — установщик скачается и запустится сам.

---

## 🔐 Лицензирование

- Каждый сотрудник EMS получает **персональный ключ** от администрации
- Ключ привязывается к устройству (HWID) при первой активации
- Передача ключей третьим лицам **запрещена**
- При смене устройства обращайся к руководству
