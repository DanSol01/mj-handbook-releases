## 📌 О приложении

**MJ Handbook** — настольный справочник для игроков на roleplay-сервера **Majestic RP**.

Работает **поверх игры** — протоколы, инструкции и заметки всегда под рукой, не выходя из игровой сессии.

---

## ✨ Что умеет

| | |
|---|---|
| 📂 **Документы** | Разделы со статьями — протоколы, инструкции, регламенты |
| 📝 **Заметки** | Свободный текст — тен-коды, позывные, прайс |
| 🔍 **Поиск** | Мгновенный поиск по всему справочнику |
| 📋 **Копирование** | Одна кнопка — текст скопирован, вставляй в чат |
| ⌨️ **Горячая клавиша** | `Ctrl+Shift+E` — скрыть/показать окно поверх игры |
| 🔄 **Автообновление** | Новые версии устанавливаются прямо в приложении |
| 💾 **Экспорт / Импорт** | Сохраняй и переноси свои данные |

---

## ⬇️ Установка

**1.** Перейди в [**Releases**](../../releases/latest) и скачай `MJ Handbook Setup x.x.x.exe`

**2.** Запусти установщик

> ⚠️ Windows может показать предупреждение — нажми **«Подробнее»** → **«Выполнить в любом случае»** (Данный справочник не имеет вирусов)

**3.** Введи лицензионный ключ, выданный администрацией
